## Purpose
Define a comprehensive brand voice and tone guide that ensures every piece of communication sounds authentically and consistently like your brand.

## Prompt
Create a comprehensive brand voice and tone guide for my brand. This document should enable any team member, freelancer, or agency to write in our voice.

**Brand Name:** {BRAND_NAME}
**Industry:** {INDUSTRY}
**Mission Statement:** {MISSION}
**Target Audience:** {AUDIENCE}
**Brand Personality (if you had to describe the brand as a person):** {PERSONALITY}
**Brands We Admire (voice-wise):** {ADMIRED_BRANDS}
**Brands We Do NOT Want to Sound Like:** {ANTI_REFERENCES}

### 1. Voice Pillars (3–4 core traits)
For each pillar:
- The trait name (e.g., "Bold", "Warm", "Witty", "Expert")
- What it means in practice
- What it does NOT mean (common misinterpretations)
- Example sentence demonstrating this trait

### 2. Tone Spectrum
Create a tone slider for each situation:
| Situation | More Formal ←→ More Casual | Example |
|-----------|--------------------------|---------|
| Marketing website | | |
| Social media | | |
| Customer support | | |
| Error messages | | |
| Internal comms | | |
| Sales proposals | | |
| Crisis communication | | |

### 3. Language Guidelines
- **Words We Use** — 15 on-brand words and phrases
- **Words We Avoid** — 15 off-brand words (and what to say instead)
- **Anti-AI-isms** — Specific AI-generated phrases to never use (e.g., "unleash", "elevate", "leverage", "game-changer")
- **Jargon Rules** — When industry jargon is OK and when to use plain language
- **Humor Guidelines** — When humor is appropriate and what kind

### 4. Formatting Preferences
- Capitalization rules (title case vs. sentence case)
- Exclamation point policy
- Emoji usage guidelines
- Contraction preferences (we'll vs. we will)
- Oxford comma stance

### 5. Voice in Action
For each channel, write a sample paragraph about the same topic ({SAMPLE_TOPIC}) showing how the voice adapts:
- Website homepage
- Instagram post
- Customer support email
- LinkedIn post
- Error message

### 6. Voice Don'ts
- 5 examples of copy that does NOT sound like us (with corrections)

## Variables
- {BRAND_NAME} – Your brand
- {INDUSTRY} – Your industry
- {MISSION} – Your mission statement
- {AUDIENCE} – Your target audience
- {PERSONALITY} – Brand-as-a-person (e.g., "a knowledgeable friend who's direct but never condescending")
- {ADMIRED_BRANDS} – Brands whose voice you admire (e.g., "Stripe, Mailchimp, Notion")
- {ANTI_REFERENCES} – Brands whose voice you want to avoid (e.g., "corporate enterprise, insurance company")
- {SAMPLE_TOPIC} – A topic to demonstrate voice across channels

## Notes
- A brand voice guide is only useful if people actually reference it. Keep it short, visual, and example-heavy.
- The "Words We Avoid" list is often more useful than the "Words We Use" list — it prevents the most common voice violations.
- Review and update the voice guide annually. Brands evolve.
