## Purpose
Write engaging short-form video scripts (TikTok, Instagram Reels, YouTube Shorts) with hooks, pacing, and CTAs optimized for the algorithm.

## Prompt
Write 5 short-form video scripts (30–60 seconds each) on the following topic. Each script should use a different viral format.

**Topic:** {TOPIC}
**Platform:** {PLATFORM}
**Target Audience:** {AUDIENCE}
**Goal:** {GOAL}
**On-Screen Talent:** {TALENT_TYPE}
**Tone:** {TONE}

### Script 1: "Things Nobody Tells You About..."
- Hook (0–3 sec): Bold statement to camera
- Body: 3–4 rapid-fire insights with visual cuts
- Close: "Follow for more {topic} tips"

### Script 2: "Stop Doing This / Do This Instead"
- Hook: "Stop doing {common mistake} RIGHT NOW"
- Split-screen or before/after format
- Show the wrong way → the right way
- Close with result/transformation

### Script 3: "POV / Day in the Life"
- Hook: Set the scene with text overlay
- Walk through a relatable scenario
- Reveal the unexpected twist or lesson
- Close: Engagement question

### Script 4: "Mini Tutorial / How-To"
- Hook: "Here's how to {desirable result} in 30 seconds"
- Step 1, Step 2, Step 3 — rapid and clear
- Each step with visual demonstration notes
- Close: "Save this for later"

### Script 5: "Myth vs. Reality"
- Hook: "Everyone thinks {myth} — here's the truth"
- Present the myth with dramatic tone
- Debunk with evidence/experience
- Close: "What other {topic} myths should I bust?"

For each script, provide:
- **Opening hook** (exact words, first 3 seconds)
- **Full script** with timestamp markers [0:00–0:03], [0:03–0:10], etc.
- **On-screen text overlay** suggestions
- **B-roll / visual cut** suggestions
- **Trending audio suggestion** (describe the audio vibe, not a specific song)
- **Hashtags** (5–8 relevant hashtags)

## Variables
- {TOPIC} – Video topic
- {PLATFORM} – Where it'll be posted
- {AUDIENCE} – Target viewer
- {GOAL} – Goal (e.g., "go viral", "drive profile visits", "promote a product")
- {TALENT_TYPE} – Who's on camera (e.g., "founder talking to camera", "hands-only tutorial", "animated text only")
- {TONE} – Vibe (e.g., "fast-paced and energetic", "calm and educational", "funny and relatable")

## Notes
- The first 3 seconds determine if someone watches or scrolls. Start with movement, a bold statement, or text on screen.
- Videos with captions/text overlays get 40% more watch time.
- Completion rate is the #1 metric for the algorithm. Keep it tight — cut everything that doesn't add value.
