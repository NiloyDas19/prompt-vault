## Purpose
Generate a comprehensive, SEO-optimized blog post outline that ranks on Google, satisfies search intent, and keeps readers engaged.

## Prompt
Create a detailed SEO blog post outline for the following target keyword. The outline should be comprehensive enough to outperform the current top 5 search results.

**Target Keyword:** {TARGET_KEYWORD}
**Secondary Keywords:** {SECONDARY_KEYWORDS}
**Search Intent:** {SEARCH_INTENT}
**Target Audience:** {AUDIENCE}
**Word Count Target:** {WORD_COUNT}
**Content Angle:** {UNIQUE_ANGLE}

Provide:

### 1. Title Tag (under 60 characters)
- Include the primary keyword near the beginning
- 3 variations: direct, curiosity-driven, and number-based

### 2. Meta Description (under 155 characters)
- Include primary keyword
- Compelling reason to click
- 2 variations

### 3. Detailed Outline
For each section, include:
- **H2 heading** (with keyword variation where natural)
- **H3 subheadings** under each H2
- **3–4 bullet points** describing what to cover in each section
- **Suggested word count** per section
- **Internal link opportunity** — what existing content to link to
- **External source suggestion** — what type of data/study to cite

### 4. Featured Snippet Optimization
- A 40–60 word paragraph answering the primary query directly (formatted for position zero)
- A comparison table if the keyword warrants it
- A numbered list or step-by-step process if applicable

### 5. E-E-A-T Signals
- Suggestions for demonstrating Experience, Expertise, Authoritativeness, and Trustworthiness
- Author bio recommendations
- Data sources to cite for credibility

### 6. Content Gaps
- What the current top 5 results are missing that you should include
- Questions from "People Also Ask" to address
- Related topics that make the article more comprehensive

## Variables
- {TARGET_KEYWORD} – Primary keyword to rank for
- {SECONDARY_KEYWORDS} – Related keywords to include naturally
- {SEARCH_INTENT} – Intent type: informational, commercial, transactional, or navigational
- {AUDIENCE} – Who's searching this
- {WORD_COUNT} – Target length (e.g., "2,000 words", "3,500 words")
- {UNIQUE_ANGLE} – Your differentiator (e.g., "based on our original data", "from a practitioner's perspective")

## Notes
- The outline IS the article. A great outline produces a great article. A vague outline produces filler.
- Always check "People Also Ask" and "Related Searches" on Google for content gap ideas.
- E-E-A-T is critical for YMYL topics (health, finance, legal). Include author credentials and cite primary sources.
