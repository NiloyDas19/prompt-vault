## Purpose
Generate stylized 2D anime or manga character illustrations with distinct Japanese animation aesthetics.

## Prompt
Anime character design of a {AGE} {GENDER} {CHARACTER_ARCHETYPE}, {HAIR_STYLE} {HAIR_COLOR} hair, {EYE_COLOR} eyes, wearing {OUTFIT_DESCRIPTION}, {EXPRESSION}, {POSE}, {BACKGROUND_SETTING}, {ANIME_STYLE} style, high quality 2D anime illustration, cel shaded, vibrant colors, key visual aesthetic

## Variables
- {AGE} – Age (e.g., "teenage", "adult", "young")
- {GENDER} – Gender (e.g., "boy", "girl")
- {CHARACTER_ARCHETYPE} – Type (e.g., "magical girl", "shonen protagonist", "mecha pilot", "transfer student")
- {HAIR_STYLE} – Haircut (e.g., "spiky wild", "long flowing", "twin tails")
- {HAIR_COLOR} – Hair color (e.g., "electric blue", "silver", "pink")
- {EYE_COLOR} – Eyes (e.g., "heterochromia", "large glowing red")
- {OUTFIT_DESCRIPTION} – Clothes (e.g., "a stylized Japanese school uniform", "futuristic combat armor", "cyberpunk streetwear")
- {EXPRESSION} – Face (e.g., "determined smirk", "shy blush", "intense battle cry")
- {POSE} – Action (e.g., "mid-air kick", "floating magically", "casual standing")
- {BACKGROUND_SETTING} – Setting (e.g., "cherry blossoms falling", "ruined city", "simple speed lines")
- {ANIME_STYLE} – Studio ref (e.g., "Studio Ghibli", "Ufotable", "Trigger", "90s retro anime")

## Modifiers
**Midjourney:** `--ar 3:4 --niji 6` (Use Niji mode specifically for anime)
**DALL-E 3:** Add "high quality anime illustration, 2d cel shaded" at start
**Stable Diffusion:** Use anime models (Anything V5, NovelAI). Negative: (3d, photorealistic, realistic, ugly, messy)

## Notes
- Using Midjourney's `--niji` parameter is essential for getting the authentic anime aesthetic rather than a westernized cartoon look.
