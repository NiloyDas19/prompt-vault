## Purpose
Generate professional character concept art suitable for game development, animation, or film preproduction.

## Prompt
Professional character concept art of {CHARACTER_NAME_AND_ROLE}, {BODY_BUILD} figure shown in {POSE_TYPE}, wearing {COSTUME_DESCRIPTION} with {MATERIAL_DETAIL} material textures, {COLOR_SCHEME} color scheme with {ACCENT_COLOR} accent, {WEAPON_OR_PROP} visible, clean white or neutral grey background, multiple annotation arrows pointing to {DESIGN_DETAILS}, {ART_STYLE} illustration style, suitable for game engine import, character turnaround sheet quality, professional {GENRE} game art

## Variables
- {CHARACTER_NAME_AND_ROLE} – Who they are (e.g., "Kira, an elite space bounty hunter", "Elder Moros, an ancient corrupted mage")
- {BODY_BUILD} – Physique (e.g., "athletic female", "stocky muscular male", "slender androgynous")
- {POSE_TYPE} – Pose (e.g., "neutral T-pose", "dynamic action pose", "confident three-quarter view")
- {COSTUME_DESCRIPTION} – Outfit (e.g., "layered tactical armor with glowing blue trim", "tattered scholar's robes with arcane symbols")
- {MATERIAL_DETAIL} – Materials (e.g., "worn leather, polished steel, luminous crystal", "silk, bone, dark magic")
- {COLOR_SCHEME} – Main colors (e.g., "midnight black and electric blue", "earthy browns and forest green")
- {ACCENT_COLOR} – Pop color (e.g., "crimson red", "gold", "neon green")
- {WEAPON_OR_PROP} – Their item (e.g., "twin pistols holstered", "a staff topped with a floating orb", "bare hands")
- {DESIGN_DETAILS} – Notable features to highlight (e.g., "facial scar, cape clasp, boot detail", "magic tattoos, gauntlet runes")
- {ART_STYLE} – Art style (e.g., "semi-realistic", "stylized cartoon", "painterly", "flat design")
- {GENRE} – Game type (e.g., "RPG", "FPS", "MOBA", "side-scroller")

## Modifiers
**Midjourney:** --ar 3:4 --v 6.1 --stylize 400
**DALL-E 3:** Add "game character concept art, professional quality" at start
**Stable Diffusion:** Negative: (background clutter, multiple characters, bad anatomy, deformed)

## Notes
- For turnaround sheet: add "three views — front, side, back — shown side by side on white background"
- For villain: add "menacing silhouette, power radiating, intimidating presence"
- Neutral/white backgrounds help with game engine import workflows — keep {POSE_TYPE} as T-pose for rigging reference
